介绍 Linux 命令行的入门书

按键/命令 功能
选中文字+鼠标中键 复制粘贴
date 显示系统当前时间和日期
cal 显示当前月份的日历
df 查看磁盘剩余空间的数量
free 显示空闲内存的数量
Alt 和 F1-F6(中的一个) 命令行→图形化
Ctrl-Alt-F6 图形化→命令行
cd ~user_name 更改工作目录到用户家目录。例如, cd ~bob 会更改工作目录到用户“bob”的家目录。

如果你按下鼠标左键，沿着文本拖动鼠标（或者双击一个单词）高亮了一些文本， 那么这些高亮的文本就被拷贝到了一个由 X 管理的缓冲区里面。然后按下鼠标中键， 这些文本就被粘贴到光标所在的位置。
